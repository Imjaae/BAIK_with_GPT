# BAIK 화면 목록

본 문서는 BAIK 버스 정류장 키오스크 서비스의 화면 명세 문서이다.  
모든 화면은 한글 화면명을 기준으로 정의되며, 디자인 이미지는 `_design` 폴더의 PNG 파일을 참고한다.

--- 전체 화면 목록 및 화면별 구성요소 

## 1. 메뉴 > 홈

### 홈화면

#### 설명
정류장을 기준으로 한 메인 홈 화면이다.  
기본 정보, 실시간 도착 버스 정보, 지도 요약, 아바타 대화창, 하단 메뉴를 제공한다.

#### 구성 요소

**헤더**
- 날짜 (년 / 월 / 일 / 요일 / 오전·오후 / 시간)
- 정류장 정보 (정류장명, 정류장 번호)
- 날씨 정보 (아이콘, 상태, 온도)
- 미세먼지 정보 (아이콘, 상태)
- 습도 정보 (아이콘, 상태)

**실시간 도착 정보**
- 현재 운행 중인 버스 번호
- 도착 예정 시간 (00분 / 곧 도착)
- 저상버스 아이콘
- 혼잡도 아이콘
- 정류장 운행 버스 전체보기 버튼

**지도**
- 지도 UI
- 지도 전체보기 버튼

**아바타**
- 대화형 아바타 UI

**푸터 메뉴**
- 홈
- 길안내
- 버스찾기
- 주변정보
- 안양시 소식
- 교통정보

---

### 홈화면 - 지도 전체보기

#### 설명
홈화면의 지도 영역을 전체 화면으로 확장한 화면이다.

#### 구성 요소
- 지도 UI (그래픽)
- 내 위치로 이동 버튼
- 카테고리 박스
- 장소 표시 마커
- 장소 상세 버튼
- 닫기 버튼

---

### 홈화면 - 전체 도착 버스 보기

#### 설명
정류장 기준으로 전체 도착 버스 정보를 한 화면에 표시한다.

#### 구성 요소
- 버스 번호
- 도착 시간
- 남은 정거장 수
- 저상버스 여부
- 혼잡도 아이콘
- 배차 간격
- 레이아웃: 3칸 × 6줄 (총 18개 표기)

---

## 2. 버스 찾기

### 버스찾기 화면 (버스번호 검색)

#### 설명
현재 정류장에 도착하는 버스를 버스 번호로 검색하는 화면이다.

#### 기능 및 구성
- 탭: 목적지 | 버스번호 (선택)
- 숫자 탭: 1 ~ 9
- 버스 목록
  - 버스 번호
  - 버스 종류
  - 가장 가까운 도착 정보
- 이전 페이지 버튼
- 다음 페이지 버튼
- 뒤로가기 버튼
- QR 버튼

- 음성 기능: 사용

---

### 버스찾기 화면 (목적지 검색)

#### 설명
목적지(정류장)를 기준으로 버스를 검색하는 화면이다.

#### 기능 및 구성
- 탭: 목적지 (기본) | 버스번호
- 초성 탭: ㄱ ~ ㅎ
- 정류장 선택 리스트 (직행 버스만 표시)
- 이전 페이지 버튼
- 다음 페이지 버튼
- 뒤로가기 버튼
- QR 버튼

- 음성 기능: 사용

---

### 버스찾기 화면 (목적지 검색 상세)

#### 설명
선택한 정류장을 기준으로 운행 중인 버스 상세 목록을 표시한다.

#### 구성 요소
- 선택된 정류장 정보
- 버스 번호
- 버스 종류
- 도착 정보 (가장 가까운 정보 1건)
- 이전 페이지 버튼
- 다음 페이지 버튼
- 뒤로가기 버튼
- QR 버튼

---

### 버스 상세 화면

#### 설명
선택한 버스 노선의 상세 정보를 제공하는 화면이다.

#### 구성 요소

**버스 기본 정보**
- 버스 번호
- 버스 노선도 (텍스트 / 이미지)
- 종점 정보
- 현재 버스 위치
- 현재 정류장 위치

**예상 시간표**
- 예상 도착 시간
- 전 정거장 수
- 혼잡 상태

**운행 정보**
- 운행 시간 (평일 / 주말)
- 배차 간격 (평일 / 주말)
- 심야버스 대체 노선 정보

**하단 버튼**
- 뒤로가기 버튼
- QR 버튼

---

## 3. 메뉴 > 목적지 찾기

### 목적지 찾기 화면

#### 설명
목적지를 다양한 방식으로 검색할 수 있는 메인 화면이다.

#### 탭 구성
- 버스정류장
- 전철역명
- 주소지 (도 / 시 / 동)
- 카테고리 (분야 > 서브분야)
- 직접입력 (초성입력)

---

### 목적지찾기 - 버스정류장 찾기

#### 설명
검색어 기반으로 버스 정류장을 찾는 화면이다.

#### 구성 요소

**제목 영역**
- “목적지 찾기 - 정류장”

**리스트 영역**
- 검색어가 포함된 버스 정류장 목록

**검색 영역**
- 이전 버튼
- 검색창 (텍스트 입력 시 키보드 표시)
- 입력 내용 삭제 버튼
- 검색 아이콘 버튼

**키보드 영역**
- 쿼티 자판
- 숫자 키패드 전환
- 한글 / 영문 토글
- 검색 버튼
- 음성 마이크 버튼

**하단 탭**
- 정류장
- 역
- 주소지
- 카테고리
- 직접입력

---

### 목적지찾기 - 전철역명 찾기

#### 설명
전철역 이름을 기준으로 목적지를 검색하는 화면이다.

#### 구성 요소
- 제목 영역 (“목적지 찾기 - 역명”)
- 지역 필터
- 전철역 목록
- 이전 페이지 버튼
- 다음 페이지 버튼

---

### 목적지찾기 - 주소지 찾기

#### 설명
도 / 시 / 동 단위로 주소지를 검색하는 화면이다.

---

### 목적지찾기 - 카테고리 찾기

#### 설명
대분류 → 중분류 → 소분류 구조로 목적지를 찾는 화면이다.

#### 예시
- 음식점 → 한식 → 식당

---

### 목적지찾기 - 직접입력

#### 설명
초성 입력을 통해 목적지를 직접 검색하는 화면이다.

---

## 4. 목적지 상세

### 목적지 상세 - 버스정류장

#### 설명
버스 정류장의 상세 정보와 도착 버스 정보를 제공한다.

#### 구성 요소
- 정류장 고유 번호
- 정류장 이름
- 정류장 방향 정보
- 전체 도착 버스 리스트
- 주변 장소 영역
- 뒤로가기 버튼
- 지도보기 버튼
- 경로안내 버튼

---

### 목적지 상세 - 전철역

#### 설명
전철역의 상세 정보 및 도착 정보를 제공한다.

#### 구성 요소
- 역 이름
- 호선 정보
- 상행 / 하행 도착 정보
- 출구 정보 리스트
- 지도 및 출구 위치 표시
- 뒤로가기 버튼
- 지도보기 버튼
- 경로안내 버튼

---

### 목적지 상세 - 카테고리

#### 설명
식당, 카페, 은행, 병원 등 장소 상세 정보를 제공한다.

#### 구성 요소
- 장소 이름
- 카테고리
- 장소 사진
- 운영 시간
- 전화번호
- 뒤로가기 버튼
- 지도보기 버튼
- 경로안내 버튼

---

## 5. 경로

### 경로목록 화면

#### 설명
현재 위치에서 목적지까지의 이동 경로 목록을 제공한다.

#### 구성 요소
- 탭: 대중교통 (기본) / 도보
- 정렬: 최단시간 / 최소비용 / 최소환승 / 최소도보
- 경로 리스트
  - 소요 시간
  - 출발 시간 ~ 도착 시간
  - 이동 수단 정보
  - 첫 대중교통 도착 예정 시간
- 뒤로가기 버튼
- QR 버튼

---

### 경로상세 화면

#### 설명
선택한 경로의 상세 정보를 한 페이지에 표시한다. (스크롤 없음)

#### 구성 요소
- 전체 소요 시간
- 전체 경로 그래픽
- 탑승 장소 / 하차 장소 정보
- 버스 번호 클릭 시 버스 상세 화면 이동
- 뒤로가기 버튼
- QR 버튼

---

- 제목

- 내용
- 뒤로가기 버튼
- QR 버튼

## 7. 메뉴 > 안양시 소식

### 소식목록

#### 설명
안양시 관련 소식을 목록 형태로 제공한다.

#### 구성 요소
- 정보 구분 (공지 / 이벤트 / 사건 / 긴급)
- 썸네일 이미지
- 제목
- 내용 요약 (3줄)
- 작성 시간
- 페이지네이션

---

### 소식상세

#### 설명
선택한 소식의 상세 내용을 제공한다.

#### 구성 요소
- 정보 구분
- 제목
- 작성 시간
- 이미지 (슬라이더)
- 내용
